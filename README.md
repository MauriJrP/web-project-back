# web-project-backend

## Prerrequisites
 * python 3 or higher installed
 * pip installed
 * mysql installed

## Installation
 * from the root folder run:
   `pip3 install -r requirements.txt`
 
## Configure your environment
 * macOS
  >$ . venv/bin/activate
 * Windows
  `py -3 -m venv venv`
  `venv\Scripts\activate`
